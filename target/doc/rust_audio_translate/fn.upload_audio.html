Expand description

Handles the upload and processing of audio files.

This function receives audio files through a form submission, uploads them to S3, transcribes the audio using AWS Transcribe, performs sentiment analysis on the transcription, and returns the job names for the processed files.

§Parameters

  • form - A Form<FileUpload<'r>> containing the uploaded audio files.
  • state - A reference to the State<Arc<Mutex<AppState>>> containing shared application state, including AWS client instances.

§Returns

A Result containing either:

  • Ok(status::Custom<Json<Vec<String>>>) with a status code of 200 (OK) and a JSON array of job names for the processed files.
  • Err(status::Custom<String>) with a status code of 500 (Internal Server Error) and an error message if any step in the process fails.
